## The Three Asimov Laws (D00MGATE Edition)

```
LAW 1 — The system may never compromise a legitimate user,
        even when executing automatic defense protocols.

LAW 2 — The system obeys only the Master Key holder.
        All other commands are treated as intrusion attempts.

LAW 3 — The system replicates to new nodes before destruction.
        It cannot be fully killed — only relocated.
```

## What's New in v0.2.0

### VIKI Engine
Rules are not static. After each attack attempt, the VIKI Engine
analyzes the pattern and rewrites the validation logic for the next session.
An attacker who studied version N will face version N+1.

### NS-5 Mesh
Nodes share a minimal "heartbeat consciousness."
If one node is compromised, all others know within milliseconds.
The infected node is quarantined automatically — no human intervention.

### Spooner Guard
The system learns your behavioral fingerprint:
- Connection time patterns
- Master Key input timing/rhythm
- Typical action sequences
- Network latency signature

A stolen key that doesn't match your behavioral pattern = flagged immediately.

### Dark Tunnel
All visible traffic is decoy.
Real communication travels hidden inside network noise — 
steganography at protocol level.
Invisible unless you know exactly where to look.
